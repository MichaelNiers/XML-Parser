## XML-Parser

A tool to read and update XML files without opening it in an editor

## Installation

`git clone "https://github.com/MichaelNiers/XML-Parser"`

`cd XML-Parser`

Compile the project and run XML-Parser.exe

## Usage

Open any XML and change it if you want
